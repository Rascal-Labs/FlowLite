FlowLite Bookmarklet Installer

Install a bookmarklet that opens AI-generated FlowLite plaintext directly in the current TEST FlowLite page.

Open target: https://rascal-labs.github.io/FlowLite/FlowLite_test.html
Open in FlowLite

On desktop, drag the “Open in FlowLite” button to your bookmarks bar. If dragging does not work, copy the bookmarklet and paste it into a bookmark URL field manually.

How to use

  1. Click Copy Bookmarklet on this page.
  2. Create a new browser bookmark.
  3. Name it Open in FlowLite.
  4. Paste the copied javascript:... code into the bookmark URL field.
  5. Copy or select FlowLite plaintext from ChatGPT or another page.
  6. Click the registered Open in FlowLite bookmark to open it in FlowLite_test.html.

Behavior

  • If the clipboard contains FlowLite syntax, the clipboard text is used first.
  • If the clipboard contains text but no FlowLite syntax, import is cancelled and only FlowLite_test.html is opened.
  • If the clipboard is empty, selected text is used only when it contains FlowLite syntax.
  • The clipboard is read only. It is not cleared or overwritten.
  • The FlowLite text is not uploaded to GitHub Pages. It is passed inside the browser with postMessage.
